# 🐔 Яйцегляд

Прототип для проекта «Контроль качества продукции на птицефабрике при помощи компьютерного зрения».

## Быстрый старт

### Запуск сервиса анализа изображений
```bash
python -m venv .venv
pip install -r requirements.txt
python main.py
```

### Запуск Frontend
```bash
cd frontend
bun install
bun run dev
```
